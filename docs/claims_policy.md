# Claims Policy

This repository is evidence infrastructure. It must not overstate Catalyst-Q capabilities.

## Allowed Before Full Runs

- "Catalyst-Q provides a public benchmark harness."
- "The harness targets SAT Competition, MaxSAT Evaluation, TSPLIB, DIMACS TSP, OR-Library, Biq Mac, QPLIB, MIPLIB, and QAPLIB style evaluations."
- "Current artifacts validate benchmark plumbing, not solver superiority."

## Allowed After Raw Runs

- "Catalyst-Q achieved X objective on Y instance under Z budget."
- "Catalyst-Q beat baseline B on metric M for suite S under protocol P."
- "Catalyst-Q had the lowest cost per validated solution on this named subset."

## Not Allowed

- Broad "solves NP" claims.
- Broad "best NP solver" claims.
- Complexity breakthrough claims in public benchmark copy.
- Any discussion of private execution internals or internal state-space mechanics.
- Any chart whose source data is not committed or linked as raw artifacts.
